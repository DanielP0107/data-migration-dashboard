# Data Migration Dashboard

## Overview
The Data Migration Dashboard is a web-based application designed to facilitate the migration of data between various sources and destinations. It provides a user-friendly interface for managing and monitoring data transfer processes.

## Features
- User authentication and authorization
- Real-time data transfer progress monitoring
- Support for multiple data sources and destinations
- Customizable data mapping
- Logging and error handling for data migration processes
- Intuitive dashboard with data visualization

## Tech Stack
- Frontend: React.js, Redux
- Backend: Node.js, Express.js
- Database: MongoDB
- Others: Docker, Kubernetes

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/DanielP0107/data-migration-dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd data-migration-dashboard
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
